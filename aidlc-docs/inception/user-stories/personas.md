## 1. ペルソナ

### 1.1 メインペルソナ: 山田 ケン（28 歳・SaaS 企業エンジニア）

| 項目 | 内容 |
|---|---|
| 職種 | バックエンドエンジニア |
| 勤務形態 | ハイブリッド（週 2 出社 / 週 3 在宅） |
| 1 日の会議 | 平均 3〜5 件、午後集中型 |
| 性格 | 集中すると周りが見えない。トイレも水分も後回し |
| 既存ツール経験 | Apple Watch でスタンドリマインダーを設定したが続かなかった |
| 痛み | 連続会議の前にトイレに行き忘れる / 退勤時に完全に消耗している / 健康管理アプリの「真面目さ」が苦手 |
| 期待 | 真面目すぎず、勝手に世話を焼いてくれる相棒が欲しい |

|---|---|

### 1.3 アンチペルソナ（ターゲット外）

- 厳格な健康管理を求める人 → MyFitnessPal 等を推奨

- 企業の管理者として従業員を監視したい人 → プロダクト憲法により拒否
